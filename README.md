## Install

`sudo pacman -S dub dmd`__
`git clone https://gitlab.com/freedom-pride-chat/silver-ore.git`

## Build

`dub build`

## Run

Run custom game "Paladin in Dark Maze of Dungeon":__
`./silve-ore paladin`__

`W,A,S,D` - movement__
`P` - attack__
`F` - toggle pet following__

Run Silver Ore:
`./silver-ore`__
`W,A,S,D` - movement__
`E,R` - Z-movement__
`M` - toggle map__
`C` - clear cache__
