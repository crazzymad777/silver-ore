## Install

`sudo pacman -S dub dmd`  
`git clone https://gitlab.com/freedom-pride-chat/silver-ore.git`

## Build

`dub build`

## Run

### Run custom game "Paladin in Dark Maze of Dungeon":  
`./silve-ore paladin`  

`W,A,S,D` - movement  
`P` - attack  
`F` - toggle pet following  

### Run Silver Ore:
`./silver-ore`  
`W,A,S,D` - movement  
`E,R` - Z-movement  
`M` - toggle map  
`C` - clear cache
